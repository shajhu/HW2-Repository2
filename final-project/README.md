# Intake-to-Note Assistant

## Overview
This project is a GenAI-powered application that converts intake form responses into structured pharmacist review notes and draft client-facing follow-ups.

## Project Scope
- One user: pharmacist
- One workflow: intake → structured note + draft follow-up
- One boundary: no diagnosis, human review required for flagged cases

## Setup Instructions
```bash
pip install -r requirements.txt
streamlit run app.py
```
